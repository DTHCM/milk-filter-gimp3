# milk-filter-gimp3

A gimp3 port of [milk-filter-gimp](https://github.com/AnalogFeelings/milk-filter-gimp/), rewritten in C.

# Installation

## Windows

Find your GIMP3 Installation directory, navigates to `plug-ins` directory, Create a `milk-filter`
directory and copy executable to `milk-filter` directory.

## Linux

Create a `~/.config/GIMP/3.0/plug-ins/milk` directory, and copy the milk executable to that directory.
